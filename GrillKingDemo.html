Grill King Restaurant App

for Android and iOS

This project was made with React Native with a few libraries like stack, drawer, and tab navigation, Axios api, React maps, React webview, DateTime Picker, firebase, and checkboxFormX. With this project I was able to view the mobile landscape with hindsight. I developed a feel for the advantages and disandvatages of native and hybrid development. Built with Expo verion: 1.0.0 VsCode Version: 1.52.1 and Node.js Version: 14.15.3. In this restaurant app you place an order, view locations, privacy policy, and login.
